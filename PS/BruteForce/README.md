# [BruteForce - 완전탐색](./BruteForce/BruteForce.md)

> 문제를 해결하기 위해 확인해야 하는 **모든 경우를 전부 탐색**하는 방법
>
> 그 중에서도 **Back-Tracking**을 통해야 하는 상황 해결하기

### 장점
- 부분점수를 얻기 좋다.

### 단점
- 전부 탐색하므로 시간복잡도가 일반적으로 높다.

## 문제 종류

>- N개 중
>  - 중복 허용(dupliceate) / 중복 없이(no duplicate)
>- M개를
>  - 순서 있게 나열(list) / 고르기(pick)
